# Classification-Titanic
В этом проекте реализован классический ML, с минимумом обработки признаков(заполнение пустых полей, отсеивание колонок с корреляцией ниже 0.1). 
Обучено и предсказано на 5 разных моделях: KNN, LogisticRegression, SVC, LinearRegression и DecisionTreeClassifier.
Датасет взят с kaggle соревнования:https://www.kaggle.com/c/titanic
